# Update Docker Setup
Updating Amboss is a simple process. For the Amboss docker setup, navigate to the command line and enter the following:

```bash
amboss-update.sh 
```

# Update Compose Setup

For the Amboss compose setup, enter the following:
```bash
docker-compose pull
docker-compose up -d --remove-orphans
```

Then, remove old images with: 
```bash
docker image prune
```
